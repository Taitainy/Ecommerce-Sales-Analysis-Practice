# Ecommerce-Sales-Analysis-Practice
This project analyzes a sample sales dataset using Microsoft Excel. It was created as part of my self-learning journey to practice data cleaning, data visualisation, basic analysis, and generating simple insights from data.
## Objective 
To analyses and create a report that shows how much the organisations has made over the years.

##Dataset used

## Questions
1.	Compare the sum of sales and profit
2.	How much profit was made from the sales of products
3.	How much sales was made from the different states?
4.	What are the top 5 products
5.	Calculate the number of orders made, the sales, profit and quantity sold
